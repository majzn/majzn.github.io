Editor Help & Reference

Shader Globals

  • iUV – normalized UV coordinates in [0,1]. Alias: U.
  • iPixel – pixel-space position in viewport space. Alias: P.
  • iFragCoord – fragment coordinate in pixels. Alias: fragCoord.
  • iTime – elapsed time in seconds.
  • iFrame – current frame count.
  • iResolution – viewport resolution as vec3 (x/y/1).
  • iMouse – mouse state as vec4.
  • iMouseDir – normalized mouse direction vector.
  • fragColor – fragment output target.
  • OUT – alias for fragColor.

Shader Helper Functions

  • VAL() – sample the current buffer at iUV.
  • SAMPLE(float dx, float dy) – sample the current buffer at offset (dx,dy).
  • SAMPLE(int dx, int dy) – integer offset sample of the current buffer.
  • READ(layerName, uv) – sample another layer by name at uv.
  • READ_OFF(layerName, uv, offset) – sample another layer with an offset.
  • READ(tex_self, iUV) – alias for VAL().
  • READ_OFF(tex_self, iUV, offset) – alias for tex_self_read_off(iUV, offset).
  • READ(tex_prev, iUV) – sample the prior frame's texture.
  • READ_OFF(tex_prev, iUV, offset) – offset sample from the prior frame.

Internal Texture Helpers

  • tex_self_read_off(uv, offset) – low-level self texture read at offset.
  • tex_prev(uv) – low-level previous-frame sample.
  • tex_prev_off(uv, offset) – low-level previous-frame offset sample.

Topology Helpers

  • wrap_plane, wrap_torus, wrap_sphere, wrap_moebius, wrap_klein, wrap_mirror, wrap_cubemap – coordinate wrap functions for texture space.
  • cubemap_area(uv) – cubemap pixel area correction.
  • cubemap_dir(uv) – cubemap direction vector from UV.

Editor Features

  • Play / Pause – toggle continuous render updates.
  • Step – advance exactly one frame.
  • Reset – reset time, frame count, and feedback state.
  • Preview Only – hide the sidebar and maximize the viewport.
  • 3D View – toggle cubemap preview mode.
  • Screenshot – capture the current viewport image.
  • Add Layer – create a new buffer layer with name, color, preset, and pass settings.
  • Snippet Library – browse and insert reusable GLSL snippets.
  • Help – open this reference panel.
  • AI Shader – generate shader code with prompt-driven AI.
  • AI Settings – configure provider, API keys, endpoint, model, and headers.
  • Editor Settings – adjust the editor theme, font, spacing, and toolbar style.
  • Undo / Redo – revert or reapply layer and code edits.
  • Export / Import – save or restore the full project state as JSON.
  • Global Resolution – force all layers to a shared width/height.
  • Per-layer Resolution – set the layer pixel size and automatic scaling.
  • Passes – control how many compute passes the layer runs per frame.
  • Layer Color – assign a UI color for each buffer layer.
  • Sidebar Toggle – show/hide layer properties and editor panels.

Layer Panel & Tabs

  • Layer Name – rename the active buffer layer.
  • Buffer Color – choose a UI color for the layer tab and controls.
  • Width / Height – set the current layer's resolution.
  • Auto – use global resolution for the layer.
  • Resolution Multiplier – scale the layer relative to the base resolution.
  • Passes – run the shader multiple times per frame for iterative effects.
  • Layer Tabs – select which buffer is active and visible.
  • Lock Preview – keep a layer pinned in the output preview.
  • Bypass / Enable – temporarily disable a layer without deleting it.
  • Remove Layer – delete a layer and its render targets.

Preset Layers

  • Empty – blank shader output.
  • White – constant white buffer.
  • Checkerboard – grid pattern using iUV.
  • Noise – procedural noise based on iUV.
  • Emboss Filter – self-convolution using READ_OFF(tex_self, iUV, offset).
  • Cached One-Pass – one-pass memory effect using tex_prev.
